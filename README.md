# AI × Web3 School 学习仓库

个人学习日志、任务证明、实验记录和 Hackathon 准备。

## 关于 AI × Web3 School

[AI x Web3 School](https://aiweb3.school/) 是面向 builders 的双语开源学习平台，系统学习 AI、区块链、智能合约、AI Agent 与去中心化 AI。

## 核心资源

- Handbook：https://aiweb3.school/zh/handbook/
- WCB 课程页面：https://web3career.build/zh/programs/AI-Web3-School
- WCB Learning 页面：https://web3career.build/zh/programs/AI-Web3-School#tab=learning
- GitHub 仓库：https://github.com/lxdao-official/aiweb3school

## 目录说明

| 目录 | 用途 |
|------|------|
| `daily/` | 每日学习日志和打卡草稿 |
| `tasks/` | 课程任务完成记录 |
| `experiments/` | 实验和代码练习 |
| `handbook-feedback/` | Handbook 问题反馈和建议 |
| `hackathon/` | Hackathon 项目准备和记录 |
