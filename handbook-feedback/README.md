# Handbook Feedback 提交指南

## 如何提交反馈

在 `handbook-feedback/` 目录下创建文件，格式如下：

### 文件命名

`YYYY-MM-DD-topic.md`，例如 `2026-05-17-network-page-typo.md`

### 反馈模板

```markdown
# 反馈标题

## 问题类型

- [ ] 错别字/语法
- [ ] 概念不清楚
- [ ] 资料过期
- [ ] 结构建议
- [ ] 补充内容
- [ ] 其他

## Handbook 页面链接

> 附上对应的 Handbook 页面 URL

## 问题描述

> 详细描述问题

## 建议改法

> 如果有建议的修改方案，写在这里

## 来源

> 问题来源（阅读、实践、讨论等）
```
